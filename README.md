# MicroVolunteer Project

Цей branch призначений для реалізації курсового проєкту MicroVolunteer згідно з технічним завданням.

## Технічне завдання

Курсовий бекенд-проєкт «MicroVolunteer» реалізує мінімальну, але повністю працездатну серверну частину сервісу, що з'єднує дві категорії користувачів:
- Уразливих людей, котрі публікують прохання про допомогу
- Волонтерів, які ці прохання виконують

## Технологічний стек

- Java 21
- Spring Boot 3.5.0
- Spring Web (REST API)
- Spring Data JPA + Hibernate
- PostgreSQL 16
- Spring Security (JWT auth)
- Keycloak 26.2.5
- Flyway (database migrations)
- Docker & Docker Compose
